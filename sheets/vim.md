# タブの移動
- gt, gT

# 範囲を選択してコピー(カット)

- コピー開始地点で "v"。ビジュアルモードに切り替え
- 任意のコマンドでコピー終了地点に移動。
- "y" (カットの場合は "d")
- 挿入先に移動
- "p"

# ファイルを開く
- :e (ひとつのファイル)
- :arg *.{c,h} (複数のファイル)

# カーソル移動
- 横: w, b, g, e 
- 特定文字に移動: f, ;
- 特定文字の前に移動: t, ;

# 削除
- df{char}: {char}までのテキストを削除
- d/={Enter}: 現在のカーソル位置から=までのテキストを削除

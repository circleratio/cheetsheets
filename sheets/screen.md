# 編集
- ^z+[: コピー開始(vi方式でカーソル移動→Space→カーソル移動→Space)
- ^z+]: ペーストする
- カーソル移動
    - C-u, C-d: 画面半分スクロール
    - C-f, C-b: ページダウン/ページアップ

# 画面分割(リージョン分割)
- ^z+S: 水平方向に分割
- ^z+|: 垂直方向に分割
- ^z+TAB: 画面を移動
- ^z+X: 画面を閉じる

# セッションのアタッチ・デタッチ
- ^z+d: デタッチ
- screen -ls: セッション一覧の表示
- screen -r <pid>: セッション再アタッチ(省略、部分マッチもOK)

# スクリーン操作
- ^z+c: 新規スクリーンの生成
- ^z+p: 前のスクリーンへ移動
- ^z+n: 次のスクリーンへ移動
- ^z+": スクリーンの一覧表示
- ^z+A: スクリーンに名前を付ける
